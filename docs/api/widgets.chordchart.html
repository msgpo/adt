adt.widgets.ChordChart

ChordChart

adt.widgets.ChordChart(name[, parent])

The chord chart widget class.
argtypedescription
namestringIdentifier of the widget.
parentobjectParent element to append widget to. If not specified, widget is appended to body. optional

radius

adt.widgets.ChordChart.radius(size)

Sets radius in pixels. Default is 100.
argtypedescription
sizenumberSize of the radius in pixels.

thickness

adt.widgets.ChordChart.thickness(size)

Sets the thickness of the segments. Default is 10.
argtypedescription
sizenumberSize of the thickness in pixels.

label

adt.widgets.ChordChart.label(text)

Adds a label below the chord.
argtypedescription
textstringLabel text.

ticks

adt.widgets.ChordChart.ticks(on)

Whether to have ticks (labels on the segments). Default is false.
argtypedescription
onbooleanWhether to have ticks or not.

colors

adt.widgets.ChordChart.colors(color)

Sets the color of the lines. Default is Tableau's standard color scheme.
argtypedescription
colorstring objectSingle color string or an object containing colors for each segment.

mouseover

adt.widgets.ChordChart.mouseover(callback)

Sets callback for mouse over on a segment. Can accept two parameters: the segment name and its index.
argtypedescription
callbackfunctionCallback to set.

mouseleave

adt.widgets.ChordChart.mouseleave(callback)

Sets callback for mouse leave on a segment. Can accept two parameters: the segment name and its index.
argtypedescription
callbackfunctionCallback to set.

invert

adt.widgets.ChordChart.invert(on)

Inverts the chord meaning the color of each chord now correspods to the largest source instead of the largest target.
argtypedescription
onbooleanIf chord should be inverted.

data

adt.widgets.ChordChart.data(data)

Binds data to the chord chart.
argtypedescription
dataArrayArray of source, target, value objects denoting the size of chord from segment to segment.

highlight

adt.widgets.ChordChart.highlight(key)

Highlights the specified segment.
argtypedescription
keystringKey of the segment to highlight.
