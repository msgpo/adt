description

A pie chart represents one dimensional categorical data of positive numbers only. It is useful when fractions of a whole (especially percentages) are to be shown for a small number of categories (5-7) and the focus is not on nuance differences among categories.

However, changes are hard to perceive on a pie chart, so use it only for static data. Furthermore, small values are difficult to distinguish or notice and therefore, avoid using pie charts when the data spans multiple orders of magnitude. Finally, due to the difficult access to small values, it should be avoided when interaction is needed.enys.
