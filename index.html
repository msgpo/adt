what is this

This is the developer site for Analytics Dashboard Tools () is a Javascript library used within Sony Mobile to quickly build interactive dashboards for displaying real-time data.

how can I use it

To use any of the modules, at the moment you need to download the module source and include it in the HTML file: 
<script src="module.js"></script>

requirements

Some of the modules rely on external libraries. Here is a list of all libraries that are used:
d3@4
SVG manipulation library. [website]
queue
Module for reading files asynchronously. [github]
topojson
Module for reading TopoJSON files. [github]
lodash
A set of useful utility methods. [website]
localforage
Module for managing local storage. [website]
spin
Widget of showing a spinner. [website]
